# Mini Issue Tracker

Laravel application for managing projects, issues, tags, and comments.

## Features

* Projects CRUD
* Issues CRUD
* Tags CRUD
* Comments on Issues
* Project → Issues relationship
* Issue ↔ Tags relationship

## Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

## Technologies

* Laravel 12
* PHP 8.2
* MySQL
